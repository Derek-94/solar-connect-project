## 멤버별 기능 구현

- 🙋‍♀️ 김효진
  
  - Timer 컴포넌트 작성, 재활용.

  - NATION 객체의 속성을 전달하여 컴포넌트 별 “ko-KR” , “en-US” 표기법 구분

  - new Date 객체를 활용하여 Timer 속성 구현

  - month, date, Component 조건부 랜더링

  - 배포

- 🙋‍♂️ 홍석기

  - Sorting component 구현
  
    - 사용자가 입력한 값은 숫자나 쉼표만 가능.

    - 그 외 입력시, 에러표기.
    
    - 소팅 시작 버튼 누르면 오름차순은 바로 나오고, 내림차순은 3초 후에 렌더링.

  - Sort algorithm

    - `Merge Sort` 사용.
